# Bondi Surf Dash

A fun HTML5 game where you help a backpacker reach the beach in Nha Trang, Vietnam, and then surf the waves!

## Game Description

Bondi Surf Dash is a two-level HTML5 game:

1. **Street Crossing Level**: Navigate through the streets of Nha Trang, dodging tuk-tuks, scooters, and street dogs to reach the beach. This level has a top-down view similar to "Frogger".

2. **Surfing Mini-Game**: Once you reach the beach, you'll hop on a surfboard for a side-scrolling endless runner. Dodge jellyfish and floating coconuts to survive for 30 seconds and win the game.

## How to Play

### Controls
- **Street Level**: Use arrow keys (↑, ↓, ←, →) to move the backpacker
- **Surf Level**: Press the spacebar to jump over obstacles

### Objective
- Street Level: Reach the beach at the top of the screen
- Surf Level: Survive for 30 seconds to win

## Running the Game

Simply open the `index.html` file in a modern web browser to start playing.

```
open index.html
```

## Game Features

- Colorful graphics with backpacker theme
- Two distinct gameplay styles
- Increasing difficulty as you progress
- Win screen with link to search for Bondi Backpackers on TripAdvisor

Enjoy your virtual trip to Nha Trang!
